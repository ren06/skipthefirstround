
1) To connect to remote client:
$ heroku pg:psql
at the prompt you can enter some SQL like 
$ SELECT * FROM tbl_user;

2) get a dump with structure only and drop and create database

$ pg_dump --host=localhost --schema-only --create --clean --format=c --no-acl --no-owner --username=admin --file "db.dump" "daf"

$ pg_dump -Fc --no-acl --schema-only --format=c --create --clean --no-owner -h localhost -U admin daf > db.dump

$ pg_dump -U admin --no-owner --no-acl --schema-only --create --clean daf -f backup.sql 

3) master data : none so far

4) run script

 heroku pg:backups restore 'https://www.dropbox.com/s/7pjtn636lfs79qj/db.dump?dl=0' DATABASE_URL

 heroku pgbackups:restore DATABASE_URL 'https://www.dropbox.com/s/7pjtn636lfs79qj/db.dump?dl=0'
